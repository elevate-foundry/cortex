01 Four Layers

Cortex is not just an inference server. It's a full semantic stack — from hardware detection to agent coordination.

Layer 1

Inference Runtime

Detects hardware, builds a tiered model hierarchy (L0–L7), routes every request to the smallest capable model. Cross-family verification catches correlated hallucinations.

Layer 2

Semantic Compression Language

Three primitives — @anchor, → verb, [scope] — compose into the wire protocol for all agent communication. Machine-parseable, human-readable.

Layer 3

Braille Encoding

256 Unicode Braille characters (U+2800–U+28FF) map 1:1 to bytes. Fixed-width fingerprints enable O(1) semantic comparison via Hamming distance.

Layer 4

Semantic State Deltas

Agents broadcast mutations, not full state. ΔS = S_t ⊖ S_{t-1}. Time-travel debugging, CRDT merge, event-sourced consensus. Git for AI thoughts.
